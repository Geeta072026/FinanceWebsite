# 📈 Graphite — Signal Analysis Agent

A sleek, interactive web-based **investment signal simulator** that analyzes stocks or funds and produces **Buy / Hold / Sell** signals with clear, explainable reasoning — all in the browser.

> ⚠️ **Note:** This project uses deterministic heuristics and simulated reasoning.  
> It does **not** fetch live market data and is **not financial advice**.

---

## ✨ Features

🔍 **Smart Analysis Agent**
- Simulates valuation, momentum, sentiment, and risk
- Long-term & short-term horizon support
- Clear Buy / Hold / Sell signal with confidence score

🎨 **Premium UI / UX**
- Dark graphite / grey theme
- Glassmorphism & subtle motion
- Smooth state transitions
- Responsive (desktop & mobile)

⚡ **Fully Frontend**
- No backend
- No APIs
- Runs locally in any modern browser

🧠 **Explainable Output**
- Confidence meter
- Factor-by-factor breakdown
- Human-readable reasoning (not generic filler)
